🚀 COMO INICIAR A API:

  1. Ativar o ambiente virtual:
     $ source venv/bin/activate

  2. Configurar .env com credenciais do NE8000:
     $ nano .env

  3. Iniciar servidor:
     $ python3 run.py
     
     OU
     
     $ make run

  4. Acessar:
     - Dashboard:    http://localhost:8000
     - API Docs:     http://localhost:8000/docs
     - Health Check: http://localhost:8000/health
